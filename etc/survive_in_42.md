#### ORM vs raw SQL
- 우선 순위는 무조건 프로젝트의 완성
  - 완성 후에 개선을 진행해야 됨
- Raw SQL 은 현시대에 딱히 해야할 이유가 없음
- ORM으로 코딩하면 언어의 종류를 맞출수가 있음
  - sql로 생각하는 뇌와 코딩언어로 생각하는 뇌가 동작이 다름
  - 테스트 코드 까지도 괜찮음
가급적 ORM을 쓰고 특정한 경우에 sql을 쓰는 것을 추천 
SQL 공부하는 것은 좋다
